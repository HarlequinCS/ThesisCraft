# Code of Conduct

This project follows a simple standard: be respectful, be precise, and do not waste other people's time.

## Expected behavior

- Keep feedback focused on the work
- Use respectful language
- Assume good intent
- Report problems clearly

## Unacceptable behavior

- Harassment
- Threats
- Personal attacks
- Deliberately misleading contributions
- Repeated spam or bad-faith edits

## Enforcement

Maintainers may remove contributions that break this standard and may decline future contributions from accounts that repeatedly ignore it.

## Scope

This code of conduct applies to repository discussions, pull requests, issues, and any other public project interaction.
