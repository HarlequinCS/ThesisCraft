# Security Policy

This repository contains a Markdown skill, not executable code.

## Supported content

- `skills/humanizer/SKILL.md`
- `README.md`
- `CONTRIBUTING.md`
- `LICENSE`

## Reporting issues

If you find a problem that could cause unsafe or misleading output, open an issue or contact the maintainer through GitHub.

Please include:

- the text that triggered the issue
- the expected behavior
- the actual behavior
- whether the text was academic, technical, or general prose

## Scope

Security concerns here are about content quality and publishing safety, not malware or network exposure.

## Response expectations

Issues that affect citation integrity, factual accuracy, or unsupported claims should be treated as high priority.
